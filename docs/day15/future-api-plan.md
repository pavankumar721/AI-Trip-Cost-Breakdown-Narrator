# Future API Plan

## POST /api/feedback

Purpose:
Allow users to submit ratings and comments for generated narrations.

Example:

{
"rating": 5,
"comment": "Very useful narration"
}

---

## GET /api/analytics

Purpose:
Provide statistics.

Examples:

* Total generations
* Average package amount
* Most selected destination

---

## GET /api/templates

Purpose:
Return available narration templates.

Examples:

* Standard
* Luxury
* Budget
* Family
